# Amazon-data-analytics-project

This project performs end-to-end data analytics on Amazon product data.

## Project Workflow

1. Data Cleaning and EDA using Python
2. SQL Database integration using PostgreSQL
3. Data Visualization using Power BI
4. Backend API using Flask
5. Frontend Dashboard using Streamlit
6. Version Control using GitHub

## Tech Stack

Python
Pandas,Numpy,Matplotlib,Seaborn
SQL - Postgresql
Powerbi
Flask Api
Streamlit

## Project Files

amazon_eda.ipynb – Data cleaning and EDA  
cleaned_data.csv – Processed dataset  
app.py – Flask backend API  
dashboard.py – Streamlit frontend dashboard  
amazon_queries.sql – SQL queries used for analysis  

## Key Insights

- Product rating distribution
- Category-wise product analysis
- Review trends
- Price analysis

## How to Run the Project

1. Run Flask backend

python app.py

2. Run Streamlit dashboard

streamlit run dashboard.py
